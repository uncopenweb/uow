User editor help

Add a user

Click Create to insert a row for a new user. Double-click the OpenID cell, type a user ID, and press Enter. Double-click the Role cell, pick a role from the drop down, and press Enter.

Change a user

Double-click a cell to edit and make your changes. Press Enter to save your edits or Escape to abort.

Delete users

Single click a row to delete. Hold down Shift or Meta and click to select additional rows. Click Delete to remove the selected rows.

Saving to the server

Rows with red text are invalid and not yet saved to the server. Valid rows immediately save to the server.
