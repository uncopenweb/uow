Audio

Speech and sound services are provided by the JSonic library.

Links

  • JSonic project page
  • JSonic documentation

Examples

Say a greeting 

uow.getAudio().addCallback(function(a) {
  a.say({text : "Welcome to the UNC Open Web Server!"});
});

Play a sound 

uow.getAudio().addCallback(function(a) {
  a.play({url : "baritone_blues"});  
});
