UI Widgets

Various reusable widgets and utilities are available for use in your projects.

API

uow.checkBrowser() → dojo.Deferred

This method checks if the user's browser supports the modern HTML5 features required by the UOW framework. If not, shows a dialog suggesting the user download Chrome or Firefox.

The return value is a dojo.Deferred. You should invoke addCallback on the deferred to register a callback function that will receive a boolean response, true if the browser is acceptable and false if not.
